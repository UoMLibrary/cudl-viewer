Starting and Stopping Foundations Viewer
----------------------------------------

Live: found-live-pres01.lib.cam.ac.uk
Dev: found-dev-pres01.lib.cam.ac.uk

1. SSH into the machine you wish to use (you may need to go via a caret machine)
2. Change user to 'root'
3. Run the command /etc/init.d/tomcat-viewer restart

You can also use the commands /etc/init.d/tomcat-viewer stop 
and /etc/init.d/tomcat-viewer start
